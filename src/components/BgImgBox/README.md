# BgImgBox

一个带有**背景图片**  默认**铺满** 的盒子
其中内容用插槽的方式插入

| 名称 | 类型(默认值) | 说明 |
| :---: | :--: | :--: | :---: |
| image | String |  图片地址 |
| size| SIZE(cover)| background-size|

## SIZE

| 名称 | 类型(默认值) | 说明 |
| :---: | :--: | :--: | :---: |
| auto |  |  按自身大小显示 |
| contain | | 将背景图放大到外边, 不拉伸和裁剪 |
| cover | | 缩放背景图片，直到它填满背景层 |
